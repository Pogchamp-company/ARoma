# ARoma

## Description

### Short
Shop

### Full
Shop powered with Golang and ReactJS

## Run
### Serve golang (api) server
```shell
cd backend-app
go mod init
go build main.go
```
### Serve node (frontend) server
```shell
cd frontend-app
npm install
npm run build
node nodeapp/app.js
```
